API KEYS FOR REPORT HARASSMENT APP
===================================

Place these files in the following locations:

1. .env                    → Project root folder
2. google-services.json    → android/app/
3. firebase_options.dart   → lib/
4. api_keys.dart           → lib/config/
5. key.properties          → android/

NOTE: The key.properties file references a keystore at:
C:\Users\amany\upload-keystore.jks

You may need to update this path or request the .jks keystore file separately.

KEEP THESE FILES CONFIDENTIAL - DO NOT COMMIT TO GITHUB!
